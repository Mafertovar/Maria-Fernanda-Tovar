La segunda semana

Visual Code es un editor de códigos y se adapta a otras interfaces. Se puede desarrollar: C, C++, C#, JAVA, JS Phyton, HYTML y PHP.

También hablamos de los distintos tipos de Line Breakers que veremos a lo largo del ciclo, acá te menciono las más importantes:

  • b: Bold, es para que aparezca marcado en NEGRITA.
  • i: Italic, es para caracteres en CURSIVA.
  • sup: Superscript, es para la descripción de orden numérico
  • br/: Realiza un salto de linea. Puede anidarse pero no tendrá un contenido como tal.
  • hr/: Resume las palabras horizontales y también para separar ideas.

      • SEMÁNTICAS

    1. Sintaxis: Reglas y regulaciones para escribir, es relacionado a la gramática.
    2. Semántica: Hace referencia a significados asociados e interpreta el programa FACILMENTE.

        3. MÁS CÓDIGOS:

      1. strong: Indica que el contenido es IMPORTANTE y le da énfasis al contenido. Este código también le da formato NEGRITA.
      2. em: También le da énfasis al contenido. A diferencia de strong, este código le da formato en CURSIVA.
      3. blockquote: Este código sirve para citas largas y genera una sangría insisible.
      4. abbr: Sirve para abreviaturas y acrónimos.
      5. cite: Cuando queremos citar libros, grabaciones o artículos. Este código le da formato en CURSIVA
      6. dfn: Este código sirve cuando se explica algo por primera vez. Algunos navegadores lo imprimen en CURSIVA (No pasa en chrome o safari)
