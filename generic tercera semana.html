Hoy hablaremos de lo que haces un CSS

CSS nos permite crear reglas que especifican la apariencia en HTML, todo está dentro de una caja y engloba los elementos o agrupaciones. Los conjuntos también pueden estar dentro de una caja.

ESTRUCTURA

  • Los elementos ahora son reglas.
  • El selector indica el elemento.
  • La declaración indica el estilo que tendrá el contenido y la diferentes declaraciones para un solo selector.
  • Las propiedadesindican los aspectos del elemento.
  • El valor especifica las configuraciones de la propiedad.
      • AQUÍ TE TRAIGO UNAS ETIQUETAS HTML QUE PODRÁS USAR EN CSS:
      1. div: Agrupa distintos elementos de HTML, junta una imagen con una lista y lo junta en cajas.
      2. span:Contiene el texto que queremos diferenciar del resto usando CSS. Suele mostrarse en mayúsculas.
      3. link: Nos redirige a una carpeta.
      4. style: Le da el código a HTML y nos permite ingresar reglas desde HTML. Casi siempre se encuentra en head y se usa para contenido pequeño.
      5. id: Nos ayuda a tener un enlace, identifica un solo elemento y este realiza el llamado de valor desde CSS
      6. class=": Brinda identificación a varios elementos que están agrupados. También se hace el llamado de valor desde CSS.
      7. Herencia: Siu un elemento se anida a otro (más pequeño), sus propiedades se plicarán a estos últimos. Si le copiaños a body, se irá a page.
