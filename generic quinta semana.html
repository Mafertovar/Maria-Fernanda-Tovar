¿Qué es un MICROCONTROLADOR?

Son chips de computadora que incluyen:

  1. Microprocesador
  2. Memoria
  3. Módulo de entrada y salida I/O

      4. También incluye:

    1. ADC
    2. Clock
    3. Ram
    4. CPU
    5. ROM
    6. I/O
    7. DAC

        8. Podemos decir que es un sofware sencillo ya que se cataloga como IDE y el entorno de desarrollo integrado hace que la tarea del programador sea más sencilla.

          También es programación orientada a objetos POO ya que:

        1. Se basa en crear un modelo del problema de destino en sus programas.
        2. Disminuye errores y promociona la reutilización del código.
